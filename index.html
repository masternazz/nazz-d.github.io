Architecture At A Glance

The short version of how the lab is operated: segmented networks, controlled ingress, identity, monitoring, and backups.

Full architecture ->
Public Services Internet -> Cloudflare -> cloudflared DMZ -> NPMplus -> approved backend

Admin Access Remote device -> Tailscale subnet router -> filtered internal VLANs
